ORM Framework doc installation

Présentation

ORM Framework est un framework MVC orienté Webservices. Il est modulaire à l'extrème, c'est à dire qu'il lui faut un minimum de dépendences pour pouvoir fonctionner. Différents conceptes sont abordés dans cette doc.

Installation

Pour installer le framework il suffit de se mêtre à l'endroit où est le fichier d'installation dans un terminal et taper : 

php ormframework_1.0.3_alpha.php mon_dossier_d_install
# pour avoir un vhost installe

ou 
php ormframework_1.0.3_alpha_test.php mon_dossier_d_install
# pour ne pas avoir de vhost installe

et suivre les instructions.
